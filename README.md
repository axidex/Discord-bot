To use this bot you need first to download this executable ffmpeg.org/download.html and create new folder "FFmpeg"

# Discord-bot

# Requirements
```no-highlight
pip install discord
pip install requests 
pip install youtube_dl
pip install PyNaCl
pip install config
```
